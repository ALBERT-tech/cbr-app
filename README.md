# CBR App

Приложение для работы с данными Центрального банка Российской Федерации.

## Описание

Это приложение позволяет получать и анализировать данные ЦБ РФ.

## Структура проекта

```
cbr-app/
├── index.html          # Основной HTML файл приложения
├── README.md           # Описание проекта
└── .gitignore          # Файлы для игнорирования
```

## Использование

Откройте `index.html` в браузере.
